## Animation & Transitions

### Hover Transitions

```css
/* Card hover */
transition: all 200ms ease;
transform: translateY(-2px);
box-shadow: 0 10px 15px rgba(0, 0, 0, 0.1);

/* Button hover */
transition: all 150ms ease;
transform: translateY(-1px);

/* Link hover */
transition: color 150ms ease;
text-decoration: underline;
```

## Notes for Implementation

1. **Maintain Consistency**: Use design tokens for all spacing, colors, and typography
2. **Touch Targets**: Ensure all interactive elements are minimum 44x44px
3. **Focus States**: Always visible for keyboard navigation
4. **Responsive Images**: Use appropriate sizes for different viewports
5. **Loading States**: Consider skeleton screens or spinners for dynamic content
6. **Error States**: Design for empty states and error messages
7. **Progressive Enhancement**: Start with HTML, enhance with CSS, add JS last
