# Accessibility Statement

This website is committed to ensuring digital accessibility for people with disabilities. We are continually improving the user experience for everyone and applying the relevant accessibility standards.

## Conformance Status

This website strives to conform to WCAG 2.1 Level AA standards. We are committed to making this site accessible to the widest possible audience.

## Features Implemented

### SEO Optimization
- ✅ **Unique Title Tags**: Every page has a descriptive, unique title tag
- ✅ **Meta Descriptions**: All pages include relevant meta descriptions
- ✅ **Open Graph Tags**: Full Open Graph implementation for social media sharing
- ✅ **Twitter Card Tags**: Twitter-specific meta tags for enhanced sharing
- ✅ **Canonical URLs**: Proper canonical URLs to prevent duplicate content
- ✅ **Structured Data**: Schema.org markup including:
  - Person schema on homepage
  - BlogPosting schema on blog articles
- ✅ **Sitemap**: XML sitemap available at `/sitemap.xml`
- ✅ **Robots.txt**: Properly configured robots.txt file

### Semantic HTML
- ✅ **Proper Landmarks**: Uses semantic HTML5 elements (`<nav>`, `<main>`, `<footer>`, `<article>`)
- ✅ **Heading Hierarchy**: Logical heading structure (h1-h6) on all pages
- ✅ **Language Declaration**: `lang="en"` attribute on HTML element

### Keyboard Navigation
- ✅ **Skip Link**: Skip to main content link appears on keyboard focus
- ✅ **Keyboard Accessible**: All interactive elements accessible via keyboard
- ✅ **Focus Indicators**: Visible focus indicators on all interactive elements (`:focus-visible`)
- ✅ **Tab Order**: Logical tab order throughout the site
- ✅ **Escape Key**: Mobile menu closes with Escape key
- ✅ **ARIA Attributes**: Proper ARIA labels and roles where needed

### Visual Design
- ✅ **Color Contrast**: All text meets WCAG 2.1 AA contrast ratios (4.5:1 for normal text, 3:1 for large text)
- ✅ **Touch Targets**: Minimum 44x44px touch target size for interactive elements
- ✅ **Responsive Design**: Mobile-friendly and responsive across all devices
- ✅ **Viewport Meta Tag**: Proper viewport configuration for mobile devices

### Images and Media
- ✅ **Alt Text**: All meaningful images include descriptive alt text
- ✅ **Decorative Images**: Font icons marked with `aria-hidden="true"`
- ✅ **Responsive Images**: Uses `<picture>` element with WebP and fallback formats
- ✅ **Lazy Loading**: Images use native lazy loading for performance

### Forms and Input
- ✅ **Labels**: All form inputs have associated labels or aria-labels
- ✅ **Focus States**: Clear focus indicators on form fields
- ✅ **Error Handling**: Accessible form validation (where applicable)

### Navigation
- ✅ **Consistent Navigation**: Consistent navigation structure across all pages
- ✅ **Current Page Indicator**: Visual and semantic indication of current page (`aria-current="page"`)
- ✅ **Mobile Menu**: Accessible mobile menu with proper ARIA attributes
- ✅ **Multiple Navigation Methods**: Multiple ways to find content (navigation, search, sitemap)

## Testing and Validation

This website has been tested with:
- **Automated Tools**: 
  - Custom accessibility audit scripts
  - Heading hierarchy checker
  - Color contrast checker
- **Keyboard Testing**: Full keyboard navigation verification
- **Semantic HTML Validation**: Proper use of HTML5 semantic elements
