# Archived Data Files

This directory contains generated report files and data from the WordPress-to-static-site migration. These files are preserved for historical reference.

## Files

### Migration Reports

- **html-update-report.json** - Report from HTML update script
  - Summary of 81 files processed, 9 image replacements
  - Generated during content migration

- **webp-conversion-report.json** - Detailed WebP conversion report (10,444 lines)
  - Records 489 images processed
  - Includes file size savings calculations
  - Complete conversion statistics

- **image-mapping.json** - WordPress URL to local path mappings (1,849 lines)
  - Maps WordPress image URLs to local file paths
  - Used during migration for URL replacement

### Download Lists

- **urls-to-download.json** - List of WordPress URLs to download
  - Contains 7 URLs (PDFs and zip files)
  - Download operations completed
