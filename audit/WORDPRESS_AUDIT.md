### 1. High Priority Items

These features/content must be migrated:

1. **All published blog posts**
   - Export format: HTML or Markdown
   - Preserve: Images, links, formatting, metadata
   
2. **All publications data**
   - Export to: JSON structure
   - Include: All metadata, links, BibTeX
   
3. **Profile information and CV**
   - Static content migration
   - Keep CV as downloadable PDF
   
4. **All media assets**
   - Download all images, PDFs
   - Optimize before migration
   - Organize in new structure

### 2. Medium Priority Items

These features should be replicated if feasible:

1. **Search functionality**
   - Implement with: JavaScript search library
   
2. **Contact form**
   - Replace with: Formspree / Netlify Forms
   
3. **Social sharing**
   - Implement with: Static social share buttons
   
4. **Comments**

```bash

wp db export wordpress-backup.sql

# List all posts

# List all pages
wp post list --post_type=page --format=json > pages.json

# List all media
wp post list --post_type=attachment --format=json > media.json
```
